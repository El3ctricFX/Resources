After Effects Shortcuts
Transform Properties Basics
Scale = S
Position = P
Opacity = T
Rotation = R
Anchor Point = A
  • Center Anchor Point = Control + Alt + Home
 Enable Time Remap = Control + Alt + T
Cut / Split = Control + Shift + D
Duplicate = Control + D
Undo = Control + Z
Redo = Control + Shift + Z
Import = Control + I
Audio Waveforms = Double Click ( L )
Create Markers = * ( Astra Key )
Text = Control + T
Adjustment Layers = Control + Alt + Y
Create Soilds = Control + Y
Render Queue = Control + M
Save Project File = Control + S
Save New Project File = Control + Shift + S
Enlarge Tabs = ` ( tilde key )
New Composition = Control + Alt + N
Play Preview without audio = . ( on # pad )
Pre-comp = Control + Shift + C

Not so useful but good to know
Create Light source = Control + alt + Shift + L
Create Camera Layer = Control + alt + Shift + C
Create Null Object = Control + Alt + Shift + Y
Most recent Effect = Control + Shift + Alt + E
Remove all effects = Control + Shift + E
Open Console = Control + F12
FX Console(Free Plugin) = Control + Space
